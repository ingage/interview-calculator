# 電卓アプリ実装課題

## 課題概要

このリポジトリには、電卓アプリのベースとなる `index.html` を用意しています。
HTML と CSS で画面の見た目はすでにできあがっているので、ご安心ください！

**やっていただきたいことは、JavaScript を追加して電卓として動くようにすることです。**

ボタンを押すと画面に数値や演算子が表示され、計算ができる状態を目指しましょう。

---

## 必須要件

以下の機能を実装してみてください。

- 数字ボタン（0〜9）を押すと、画面（表示エリア）に反映される
- 四則演算（`+`, `-`, `×`, `/`）が正しく動作する
- かっこ `()` を使った計算ができる
- クリアボタン（C）で入力がリセットされる
- 1文字削除ボタン（←）で直前の入力を削除できる
- イコールボタン（=）で計算結果が表示される

---

## 任意課題（加点項目）

必須要件が実装できたら、余裕があれば以下の機能にもぜひチャレンジしてみてください！

## 提出方法

1. このテンプレートリポジトリから **「Use this template」** ボタンを押して、自分の GitHub アカウントにリポジトリを作成してください
2. 課題を実装したら、GitHub に push しましょう
3. GitHub Pages を使って HTML を公開してください（手順は下にまとめています）
4. 最後に、以下の **2つの URL** を提出してください
   - GitHub リポジトリの URL
   - GitHub Pages の URL

---

## GitHub Pages の公開手順

1. GitHub 上で自分のリポジトリページを開きます
2. **Settings** タブをクリックします
3. 左メニューから **Pages** を選びます
4. 「Source」で **Deploy from a branch** を選択します
5. Branch を **main**、フォルダを **/ (root)** に設定して、**Save** をクリックします
6. 数分ほど待つと、表示される URL からページが公開されます

詳しくは GitHub Pages の公式ドキュメントも参考にしてみてください。
