# 電卓アプリ実装課題

## 課題概要

このリポジトリには、電卓アプリのベースとなる `index.html` が用意されています。
HTML と CSS で画面の見た目はすでに実装済みです。

**あなたの課題は、JavaScript を追加して電卓として動作するようにすることです。**

ボタンを押すと画面に数値や演算子が表示され、計算が実行できる状態を目指してください。

---

## 必須要件

以下の機能を実装してください。

- 数字ボタン（0〜9）を押すと、画面（表示エリア）に反映される
- 四則演算（`+`, `-`, `×`, `/`）が正しく動作する
- かっこ `()` を使用した計算ができる
- クリアボタン（C）で入力がリセットされる
- 1文字削除ボタン（←）で直前の入力を削除できる
- イコールボタン（=）で計算結果が表示される

---

## 任意課題（加点項目）

必須要件を満たした上で、以下の機能にも挑戦してみてください。

- 税率を設定し、税込・税抜の計算ができる機能
- 商と余りを表示する機能
- その他、自分で考えた独自の機能（加点対象）

## 提出方法

1. このテンプレートリポジトリから **「Use this template」** ボタンを押し、自分の GitHub アカウントにリポジトリを作成してください
2. 課題を実装し、GitHub に push してください
3. GitHub Pages を使って HTML を公開してください（手順は後述）
4. 以下の **2つの URL** を提出してください
   - GitHub リポジトリの URL
   - GitHub Pages の URL

---

## GitHub Pages の公開手順

1. GitHub 上で自分のリポジトリページを開く
2. **Settings** タブをクリック
3. 左メニューから **Pages** を選択
4. 「Source」で **Deploy from a branch** を選択
5. Branch を **main**、フォルダを **/ (root)** に設定し、**Save** をクリック
6. 数分待つと、表示される URL からページが公開されます
